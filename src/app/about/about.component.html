About Carteado

Carteado is a free, open-source Planning Poker tool for estimating work with development teams. Create rooms, invite your team, and vote in real time.

How it works

  1. Create a room or join with an existing code
  2. Share the room link or code with your team
  3. Each participant selects a card to estimate the task
  4. The admin reveals the votes and the team discusses the estimate
  5. Start a new round for the next task

Features

  • ✓ Real-time synchronization via WebSocket
  • ✓ Customizable decks (Fibonacci, numbers, emojis)
  • ✓ Spectator mode for observers
  • ✓ Voting history by round
  • ✓ Summary with average, min, max and sum of votes

Technologies

Angular, Tailwind CSS, WebSocket, Node.js
